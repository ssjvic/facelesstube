CONTEXTO:
---------
FacelessTube tiene 2 versiones de distribución:
1. Play Store → cobra a través de Google Play Billing (30% comisión Google)
2. Web/APK (descarga directa) → cobra a través de Stripe directo (~2.9% + $0.30)

RAZONAMIENTO:
-------------
- Google exige que si distribuyes en Play Store, DEBES usar Google Play Billing
- Para la versión web/APK descargable, Stripe es más rentable (2.9% vs 30%)
- Mantener las 2 versiones permite maximizar ganancias según el canal
- El código ya está preparado: playBilling.js para Android, stripe.js para web

====================================================================
SOBRE LA RESTRICCIÓN POR IP:
====================================================================

Stripe SÍ permite restringir API keys por IP, pero SOLO desde el Dashboard:
1. Ve a Dashboard > Developers > API Keys
2. Click en la Secret Key (o la que quieras restringir)
3. Busca "Restrict key" o "IP restrictions"  
4. Añade tu IP pública
5. Para saber tu IP: busca "what is my ip" en Google

NOTA: Esto solo aplica para las API keys (backend). La publishable key
(pk_test) no necesita restricción porque es pública por diseño.
